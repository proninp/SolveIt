# [Бонусы сотрудников](https://solvit.space/coding/3210)

В базе данных используются две таблицы:
* `employees` — информация о штате компании.
* `bonuses` — информация о выданных бонусах.
Напишите SQL-запрос, который вернет имя и сумму бонуса для каждого сотрудника, чей бонус меньше 1000 или вовсе отсутствует.

Верните таблицу результатов в любом порядке.
