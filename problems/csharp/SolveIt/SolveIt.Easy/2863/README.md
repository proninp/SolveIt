# [Валидный пароль](https://solvit.space/coding/2863)

Напишите функцию для определения валидности пароля.
Дана строка `password`, верните `true`, если она является валидным паролем, или `false` в противном случае.

Условия валидности:
* 1) Только символы английского алфавита
* 2) Минимум 1 цифра
* 3) Минимум 1 допустимый символ (_, #, %)
* 4) Минимум 8 символов
* 5) Минимум 1 заглавная буква
* 6) Не является палиндромом (при любом регистре)


**Пример 1:**
Ввод: `Abc1_def#`
Вывод: `true`
Объяснение: пароль `Abc1_def#` удовлетворяет всем условиям


**Пример 2:**
Ввод: `abcdefg1_`
Вывод: `false`
Объяснение: пароль `abcdefg1_` не содержит заглавной буквы


**Пример 3:**
Ввод: `ABCDEFG1_`
Вывод: `true`
Объяснение: пароль `ABCDEFG1_` удовлетворяет всем условиям
