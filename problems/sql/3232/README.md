# [Доля заказов со скидкой](https://solvit.space/coding/3232)

Используя таблицу `orders` необходимо написать SQL-запрос, чтобы вычислить долю заказов с промокодами.

Округлите результат (`promo_orders_percentage`) до двух знаков после запятой.

Пример результирующей таблицы:

`promo_orders_percentage`
`75.00`
