# [Угол между стрелками часов](https://solvit.space/coding/2866)

При наличии двух чисел `hour` (часы) и `minutes` (минуты),
вернуть наименьший угол (в градусах), образованный между часовой `hour` и минутной `minute` стрелками.
Ответы в пределах 10^-5 фактического значения будут приняты как правильные.

**Пример 1:**

![clock01](./res/001.png)

Ввод: `hour = 12, minutes = 30`
Вывод: `165`


**Пример 2:**

![clock02](./res/002.png)

Ввод: `hour = 3, minutes = 30`
Вывод: `75`


**Пример 3:**

![clock03](./res/003.png)

Ввод: `hour = 3, minutes = 15`
Вывод: `7.5`

**Ограничения:**
* 1 <= hour <= 12
* 0 <= minutes <= 59
