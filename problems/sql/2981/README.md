# [Расчёт выручки магазина](https://solvit.space/coding/2981)

Есть две таблицы:

* таблица продаж (`sales`), в которой содержится информация о дате продажи `datetime`, магазине `shop`, артикуле `art`, который продали, и количестве проданных штук данного артикула `quantity`,
* вторая таблица (`prices`) - справочник цен с информацией по ценам `price` всех артикулов `art`.


Напишите запрос, который посчитает сумму продаж по магазину с `id=100` за 1 января 2024 года.

Поля в результирующей таблице: `total_revenue`.

Количество проданных штук может быть отрицательным для возвратов.
