# [Бинарный поиск](https://solvit.space/coding/3212)

Дан массив целых чисел nums, отсортированный в порядке возрастания, и целое число `target`.

Напишите функцию для поиска `target` в `nums`. Если `target` существует, верните его индекс. 
В противном случае верните `-1`.

**Пример 1:**

Ввод: `nums = [-1,0,3,5,9,12], target = 9`
Вывод: `4`
**Объяснение:** 9 существует в массиве nums и его индекс равен 4.

**Пример 2:**

Ввод: `nums = [-1,0,3,5,9,12], target = 2`
Вывод: `-1`
**Объяснение:** 2 не существует в массиве nums, поэтому возвращаем -1.

**Ограничения:**
* `1 <= nums.length <= 10^4`
* `-10^4 <= nums[i], target <= 10^4`
* Все целые числа в nums уникальны.
* nums отсортирован в порядке возрастания.
