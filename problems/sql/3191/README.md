# [Актуальное состояние товаров](https://solvit.space/coding/3191)

Даны таблицы:
* `items` - таблица истории цен.
* `orders` - журнал транзакций.
Напишите запрос, который выведет актуальное состояние товаров на `17 декабря 2025 года`.

Поля в результирующей таблице: `item_id`, `name`, `price`, `update_date`.

Пример результирующей таблицы:

| item_id | name        | price | update_date |
|----------|-------------|-------|--------------|
| 6002     | Блокнот А6  | 89    | 2025-09-02   |
| 2012     | Карандаш 2B | 20    | 2025-12-17   |
